📖 Setup Instructions (expand)
Step 1 — Firebase (Google Sign-In)
1. Go to console.firebase.google.com → Create project
2. Add a Web App, copy the config object
3. Go to Authentication → Sign-in method → Enable Google
4. Add code-manigopal.github.io to Authorized Domains
5. Paste the config into FIREBASE_CONFIG in this file

Step 2 — GitHub Personal Access Token
1. GitHub → Settings → Developer Settings → Personal Access Tokens → Fine-grained
2. Repository: code-manigopal.github.io only
3. Permissions: Contents → Read & Write
4. Copy the token into GITHUB_TOKEN

Step 3 — Deploy
Push this index.html into your repo as e.g. /tobermory/index.html
Access at: https://code-manigopal.github.io/tobermory/

Step 4 — Customise members
Edit the DEFAULT_MEMBERS array at the top of the script with your group's real names.

⚠️ Security note: The GitHub token is visible in the HTML source. For a private trip group this is fine. For public use, move the token to a Cloudflare Worker or similar backend proxy.
