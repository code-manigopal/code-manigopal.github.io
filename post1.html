Installing/Deploying Jenkins in Linux Environment

1. Sign in as root User. (sudo su - / <root password>)
2. export "http_proxy" and "https_proxy"
3. wget -O /etc/yum.repos.d/jenkins.repo http://pkg.jenkins.io/redhat-stable/jenkins.repo 
4. rpm --import http://pkg.jenkins.io/redhat-stable/jenkins.io.key
5. yum install jenkins java-1.8.0-openjdk â€“y
6. It will ask for downloading packages. Enter 'y'. So that packages will be downloaded and     installed. 
7. systemctl start jenkins
8. systemctl enable jenkins
9. Open the browser and put http://<ip-Address-of-your-system>:8080 
10. vi /var/lib/jenkins/secrets/initialAdminPassword 
11. Copy the Password and paste the password in the Admin Password Text Box. 
12. Configure Proxy - "<proxy-host>:<port>".
13. Install Suggested Plugins. Jenkins will install all the plugins which are required. 
14. Create First Admin Username and Password and Enter Full Name & email Address.
15. Click Save and Continue. 
16. Instance Configuration URL will be shown. Click Save and Finish. 
17. Jenkins is ready. Start Using Jenkins.
